# Canada Coin - Plataforma de Gestión de Inversiones

## Descripción
Canada Coin es una plataforma web avanzada para la gestión de inversiones, desarrollada con tecnologías modernas y estándares de seguridad financiera.

## Características Principales
- ✅ Sistema de autenticación seguro con Firebase Auth
- ✅ Dashboard financiero interactivo
- ✅ Gestión de depósitos y retiros
- ✅ Subida segura de comprobantes
- ✅ Sistema de referidos con bonificaciones
- ✅ Soporte multiidioma
- ✅ Interfaz responsive y moderna
