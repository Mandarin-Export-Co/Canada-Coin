# Assets - Imágenes y Recursos Visuales

Esta carpeta contiene todos los recursos visuales para Canada Coin.

## Estructura recomendada:

### Logos y Branding
- `logo.png` - Logo principal (200x50px)
- `logo-icon.png` - Icono de la app (64x64px)
- `favicon.ico` - Favicon del sitio

### Ilustraciones
- `dashboard-hero.png` - Ilustración del dashboard
- `empty-state.png` - Ilustración para estados vacíos
- `success-state.png` - Ilustración para operaciones exitosas

### Iconografía
- `payment-methods/` - Iconos de métodos de pago
  - `bank-transfer.png`
  - `paypal.png`
  - `bitcoin.png`
  - `ethereum.png`

### Documentación
- `screenshots/` - Capturas para documentación
- `wireframes/` - Diseños y mockups

## Especificaciones técnicas:

- Formatos: PNG, SVG, JPG (optimizados)
- Tamaño máximo: 500KB por imagen
- Resolución: 2x para pantallas retina
- Colores: Coherentes con la paleta de la marca

## Paleta de colores Canada Coin:

- Primario: #3B82F6 (Azul)
- Secundario: #1E40AF (Azul oscuro)
- Éxito: #10B981 (Verde)
- Advertencia: #F59E0B (Ámbar)
- Error: #EF4444 (Rojo)
- Neutral: #6B7280 (Gris)
